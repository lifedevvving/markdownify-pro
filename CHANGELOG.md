## [1.0.0] - 2025-01-01

### Added
- HTML to Markdown conversion with Turndown
- DOCX to Markdown via Mammoth
- Batch directory processing
- URL fetch and convert
- `.markdownify.json` configuration support
- GFM table support
- Smart noise stripping (scripts, ads, nav)
